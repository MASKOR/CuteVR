### Prerequisites
* CMake 3.5.1 (3.9.x is recommended)
* GCC 5.4 (6.0 is recommended)
* Qt 5.5.0 Core and GUI
* SteamVR

###### Additional packages for examples
* ROSPublisher
  * ROS base
* SolarSystem
  * Qt 5.5.0 OpenGL
* ViveMonitoring
  * Qt 5.5.0 QML
